# Lingling 网站

欢迎来到 Lingling 项目！这是一个现代化的基础网站，展示了如何使用 HTML、CSS 和 JavaScript 创建专业的网页。

## 📁 项目结构

```
lingling/
├── index.html       # 首页
├── about.html       # 关于页面
├── services.html    # 服务页面
├── contact.html     # 联系页面
├── styles.css       # 样式表
└── README.md        # 本文件
```

## 🚀 功能特性

- **响应式设计**: 在所有设备上都能完美显示
- **现代化界面**: 美观的用户界面和流畅的交互体验
- **导航菜单**: 简单易用的导航系统
- **联系表单**: 功能完整的联系表单
- **SEO 友好**: 优化的元标签和结构

## 📄 页面说明

### 首页 (index.html)
- 展示品牌信息
- 突出主要特性
- 调用行动按钮

### 关于页面 (about.html)
- 公司使命和价值观
- 公司承诺

### 服务页面 (services.html)
- 展示所有提供的服务
- 网格布局展示

### 联系页面 (contact.html)
- 联系表单
- 联系信息（电话、邮箱、地址）

## 🎨 样式特点
